The Problem

Public blockchain payroll destroys companies from the inside.

  • Every salary is visible — employees discover compensation gaps, morale collapses
  • Competitors see your burn rate and poach talent by outbidding exact salaries
  • Every payout creates a permanent employer-to-employee link on-chain
  • Workers earn every second but get paid every 30 days

On a public blockchain, compensation becomes office gossip.

Speaking Notes

The core problem is that on public blockchains, every transaction is visible. When you pay someone, anyone can see who paid whom and how much. For payroll, this is catastrophic. Employees discover salary gaps and get resentful. Competitors reverse-engineer your org chart and poach talent. And the traditional 30-day pay cycle forces a liquidity gap on workers.

Speaking Notes

Solana provides the base layer with sub-second finality and near-zero transaction costs. MagicBlock TEE stores private payroll state in Ephemeral Rollups. Autonomous cranks schedule checkpoint accrual without manual ticks. Private transfers use delay/split via MagicBlock Payments API to break traceability. MongoDB stores metadata, history, and compliance events. JWT auth tokens with expiry checking secure TEE state access. Fully on-chain with keeper-free design.

How It Works

For Employers

Connect wallet → Create employee → Create stream with rate → Onboard to PER (4 transactions) → Schedule checkpoint crank → Salary accrues autonomously.

For Employees

Connect wallet → Initialize private account (one-time) → View live accrual from TEE → Request cashout → Claim payout via private transfer.

Autonomous Streaming

Employer schedules checkpoint crank → MagicBlock TEE advances payroll state every second → Employer triggers settlement → Private transfer with delay/split obfuscation.

Speaking Notes

For employers: connect wallet, create employee record, create stream with rate per second, onboard to PER via 4 transactions (createEmployee, createPermission, delegate, initializePrivatePayroll), then schedule checkpoint crank. Salary accrues autonomously without manual ticks. For employees: connect wallet, initialize private account once, view live accrual from TEE, request cashout, claim via private transfer. The autonomous streaming flow uses MagicBlock crank for continuous accrual, employer triggers settlement, and private transfers with delay/split break traceability.
